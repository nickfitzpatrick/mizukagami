Leading the data pipeline that feeds the generative models: ingesting
drawing files, normalizing geometry, extracting features (room areas,
adjacencies, circulation). Storing structured output in Postgres. The
pipeline needs to be robust to messy input files since architects export
from many different CAD tools. This is the backend plumbing behind both the
generative design and compliance features.
